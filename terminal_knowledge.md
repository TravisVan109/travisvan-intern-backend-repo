# Terminal Knowledge – Reflection

## Terminal Client I Chose
I chose **Windows Terminal** because:
- It is already available on Windows
- It supports multiple shells (PowerShell, Command Prompt, WSL)
- It is fast, modern, and easy to customize
- It works well with development tools and Git


## Customizations I Made
I kept the setup simple, but I made a few useful tweaks:

### customizations 
+ Changed the theme: I selected a darker theme to reduce eye strain during long sessions.
+ Set PowerShell as my default profile  
+ Added a couple of basic aliases (just for practices, then I removed them)
For example:
- `gs` → `git status`
- `gp` → `git pull`

## Most Useful Command I Learned
The most useful command I learned today was:

### **`cd` — Change Directory**
This command lets me navigate through folders directly from the terminal.

Example:
cd Documents\Projects\travisvan109
