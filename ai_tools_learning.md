# AI Tools Learning – Claude Code

## Tool I Used
I used **Claude Code** for this task. It is an AI coding assistant that can read files, explain code, and help with debugging.

## Task 1: Understanding Existing Code

### What I Did (Simple Example)
I copied a small function from my project (a basic calculator function) and asked Claude:

> “Can you explain what this function does in simple terms?”

### What Claude Did
- It explained the purpose of the function  
- Described what each line does  
- Pointed out how the inputs and outputs work  

### What Helped
- Quick explanation  
- Easier to understand the code structure  

### What Didn’t Work
- It assumed extra details not in my code  


## Task 2: Debugging a Small Issue

### What I Did (Simple Example)
I pasted a piece of code that caused an error and asked:

Why is this error happening and how can I fix it?

### What Claude Did
- Identified the variable that was undefined  
- Suggested the correct fix  
- Explained the reason for the error  

### What Helped
- Clear steps to solve the bug  
- Good learning explanation  

### What Didn’t Work
- The first suggestion didn’t fully fix it, so I had to adjust it myself  


## Summary: What Claude Code Helped With
- Understanding small pieces of code quickly  
- Debugging simple errors  
- Giving explanations that help learning  
- Saving time on small tasks  


## Summary: Where It Struggled
- Needed more context for bigger problems  
- Sometimes guessed parts of the code  


## When AI Is Most Useful
- When I need help understanding code  
- For simple debugging  
- For quick explanations  
- For learning new concepts  

AI is helpful for support, but I still need to double-check everything.
