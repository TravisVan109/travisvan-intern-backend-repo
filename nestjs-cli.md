#  Using NestJS CLI for Scaffolding
## How does the NestJS CLI help streamline development?
The CLI automates creation of modules, controllers, and services. It sets up files with proper structure, imports, and decorators, saving time and reducing human mistakes.

## What is the purpose of nest generate?
nest generate (or nest g) creates NestJS components like modules, controllers, services, guards, pipes, and interceptors. It helps maintain a consistent and organized project structure.

## How does using the CLI ensure consistency across the codebase?
Because the CLI follows NestJS best practices, all generated files use the same patterns, naming conventions, and architecture, making the codebase easier to read and maintain.

## What types of files and templates does the CLI create by default?
The CLI can create modules, controllers, services, guards, interceptors, filters, pipes, gateways, middleware, and complete CRUD resources. Each comes with boilerplate code and correct decorators.
